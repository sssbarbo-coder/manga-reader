# Как запустить сайт в интернете (Бесплатно)

Чтобы открывать вашу читалку с телефона через интернет (а не только через домашний Wi-Fi), вам нужно «захостить» проект на одной из бесплатных платформ. Самый простой и современный способ — **Vercel** или **Netlify**.

## Вариант 1: Vercel (Рекомендуется)

1. **Создайте аккаунт на [GitHub](https://github.com)** (если его нет).
2. **Создайте новый репозиторий** и загрузите туда ваш код (папку `manga-web`).
3. **Зайдите на [Vercel.com](https://vercel.com)** и авторизуйтесь через GitHub.
4. Нажмите **"Add New" -> "Project"**.
5. Выберите ваш репозиторий с мангой.
6. В настройках сборки всё подставится автоматически (Vite определяется сам). Нажмите **"Deploy"**.
7. Через минуту вы получите публичную ссылку вида `manga-web.vercel.app`, которая будет работать везде!

## Вариант 2: Netlify

Процесс почти такой же, как с Vercel:
1. Зайдите на **[Netlify.com](https://netlify.com)**.
2. Выберите **"Import from Git"**.
3. Подключите ваш GitHub и выберите репозиторий.
4. Нажмите **"Deploy site"**.

## Вариант 3: Локальный туннель (Временный способ)

Если вы не хотите ничего загружать на GitHub, можно использовать сервис **localtunnel**, который создаст временную публичную ссылку для вашего запущенного сервера:

1. Не выключая запущенный `npm run dev`, откройте второй терминал.
2. Выполните команду (нужно будет подтвердить установку):
   ```bash
   npx localtunnel --port 5173
   ```
3. Вы получите ссылку вида `https://unique-id.loca.lt`.
4. Откройте её на телефоне. (При первом входе вас могут попросить ввести ваш внешний IP — его можно узнать на сайте `2if.ru`).

---
**Совет**: Вариант с **Vercel** — самый надежный. Сайт будет работать 24/7, даже если ваш компьютер выключен.
