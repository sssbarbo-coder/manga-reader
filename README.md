# Manga Premium Web App (PWA)

Премиальное веб-приложение для чтения манги, оптимизированное для мобильных устройств. 
Использует API MangaDex для получения актуальных данных.

## Стек технологий
- **Frontend**: React + TypeScript + Vite
- **Стилизация**: Tailwind CSS (Premium Dark Theme + Glassmorphism)
- **Анимации**: Framer Motion
- **Иконки**: Lucide React
- **Роутинг**: React Router v6

## Основные возможности
- 📱 **Mobile-first дизайн**: Идеально выглядит на смартфонах.
- 🌙 **Премиальная темная тема**: Глубокий черный фон (#0B0B0F) и эффект стекла.
- 🔍 **Поиск**: Удобный поиск по библиотеке MangaDex с дебаунсом.
- 📖 **Читалка**: Бесконечный вертикальный скролл с ленивой загрузкой страниц.
- ⭐️ **Избранное**: Сохранение любимой манги в локальное хранилище.
- 🕒 **История**: Автоматическое сохранение прогресса чтения.
- 📲 **PWA**: Возможность установки на главный экран смартфона.

## Как запустить

1. Убедитесь, что у вас установлена **Node.js 18+**.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```
4. Откройте ссылку в браузере. Для тестирования мобильного вида используйте режим разработчика (F12 -> Mobile View).

## Структура проекта
- `src/services/mangadex.ts` — Слой работы с API.
- `src/store/` — Глобальное состояние (Избранное, История, Настройки).
- `src/components/reader/` — Компоненты для чтения.
- `src/pages/` — Страницы приложения.
